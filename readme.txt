ส่วนที่ 1 โครงสร้างไฟล์ในโฟล์เดอร์โปรเจกต์
1.1 index.php - หน้าแรกของโปรเจกต์ที่ใช้สำหรับลงทะเบียนและเข้าสู่ระบบเพื่อเข้าเล่นเกม
1.2 landingPage.php - หน้าที่ใช้ในการเลือกภาษาของเกมที่จะเล่น
1.3 lang_C.js - ไฟล์ JavaScript ที่เป็นส่วนหนึ่งของการเล่นภาษา C
1.4 lang_C.js - หน้าที่แสดงคะแนนเมื่อผู้เล่นเสร็จสิ้นการเล่นเกมทุกข้อ
1.5 lang_C.js - หน้าที่ใช้ในการเริ่มเล่นเกมใหม่ โดยผู้เล่นสามารถกดปุ่ม restart เพื่อเข้าเล่นใหม่ได้
